# Tutorials d'Òmiques

Aquesta subsecció recull tutorials pràctics per facilitar l'anàlisi de dades òmiques dins del marc de treball del Servei de Bioinformàtica de l'IRIS-CC.

## Continguts previstos

- Introducció a fluxos d'anàlisi de dades genòmiques.
- Bones pràctiques en transcriptòmica (QC, normalització i interpretació).
- Bases de proteòmica i metabolòmica orientades a projectes biomèdics.
- Integració multiòmiques i visualització de resultats.
- Guia de reproduïbilitat i documentació de pipelines.

## Format dels tutorials

Cada tutorial inclourà:

1. Objectiu i context del cas d'ús.
2. Requisits (dades, software i recursos computacionals).
3. Passos detallats amb exemples.
4. Resultats esperats i criteris de qualitat.
5. Recomanacions per a la interpretació biològica.
